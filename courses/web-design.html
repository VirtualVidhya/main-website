Web Design

7-12 months

Overview

Dive into the dynamic world of Web Development & Design, where creativity meets functionality. Our comprehensive course equips students with the essential skills to craft visually stunning and user-friendly websites. From mastering HTML, CSS, and JavaScript to exploring responsive design and UX principles, students will learn the tools and techniques needed to bring their digital visions to life. Whether you're aspiring to become a front-end developer, UX/UI designer, or full-stack engineer, this course provides the perfect foundation for launching a successful career in the ever-growing field of web development.

Course Curriculum

  • Basic understanding of the Internet
  • Introduction to web, webpages, websites
  • HTML5 Introduction, tags & elements
  • HTML lists, forms
  • CSS3 Introduction & Basic understanding of styling
  • CSS dimensions & spacing
  • Basic designing with text formatting, borders, links, lists, table styling and backgrounds
  • Understanding of elements structure with display & box model
  • Understanding of web page layouts using flexbox & grid
  • Responsive web design using web page layouts & media queries
  • Proper understanding of good UI/UX
  • Use of modern libraries - Wordpress & TailwindCSS
  • Basics of programming with JavaScript
  • Dynamic websites by manipulating HTML & CSS using JavaScript
