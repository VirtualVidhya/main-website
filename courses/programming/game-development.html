Game Development

7-12 months

Overview

Our Game Development course is designed for aspiring game creators who want to bring their imaginative worlds to life. Through hands-on projects, students will learn the fundamentals of game design, level design and game programming. From crafting compelling narratives to mastering advanced coding techniques, this course empowers students to create immersive gaming experiences across various platforms. By the end of the program, students will have the skills and confidence to develop their own games, ready to embark on a journey in the exciting and ever-evolving world of game development.

Course Curriculum

  • Basics of programming in C++
  • Fundamentals of Object Oriented Programming in C++
  • Introduction of Game Development, Industry, Different Roles
  • Fundamentals of Unity & C#
  • Fundamentals of Unity UI & practices for good UI/UX in games based on platform
  • Unity UI Events & Input Handling
  • Unity Animations
  • Sound Effects & Audio Management across the game
  • Unity 2D physics
  • Unity 3D physics
  • Basic Enemy Artificial Intelligence
  • Camera Movement in Unity
  • Fundamentals of Version Control System with Git using Github
  • Publishing Game Build for PC & Mobile devices
  • Basic opimtimization and performance enhancements with focus on good code quality
